How to install:

Launch Eclipse,
Click on ""Help > Install New Software"
Click on button Add to add an new repository,
Enter name as "QuickShell" and enter location as https://nileshkhaire.github.io/QuickShell/update, then click on button OK,
Check QuickShell,
Follow instructions,
restart.

More details on home page : https://nileshkhaire.github.io/QuickShell/
